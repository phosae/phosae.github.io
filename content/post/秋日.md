<br/>
<br/>
<br/>
主呵，是时候了。夏天盛极一时。 <br/>

把你的阴影置于日晷上， <br/>

让风吹过牧场。<br/>
<br/> 
<br/> 
让枝头最后的果实饱满； <br/>

再给两天南方的好天气，<br/>

催它们成熟，把 <br/>

最后的甘甜浓酒。 <br/>
<br/>
<br/> 
谁此时没有房子，就不必建造，<br/>

谁此时孤独，就永远孤独， <br/>

就醒来，读书，写长长的信， <br/>

在林荫路上不停地 <br/>

徘徊，落叶纷飞<br/>
